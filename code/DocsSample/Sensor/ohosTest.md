# 数据防泄漏测试用例归档

## 用例表

| 测试功能    | 预置条件                | 预期输出                          | 是否自动                |测试结果|
|---------|---------------------|-------------------------------|--------------------------------|--------------------------------|
| 拉起应用    | 	设备正常运行             | 成功拉起应用                        | 是                 |Pass|
| 输入数据采样  | 自动输入100000000       | 拉起输入页面，输入100000000 | 是            |Pass|

| 订阅传感器类型 | 持续订阅传感器，数据正常上报	     | 成功订阅accelerometer传感器，持续上报数据 | 是 |  Pass|

| 订阅一次传感器 | 	 上报订阅传感器一次数据    | 订阅一次传感器成功，上报一次数据   | 是 |  Pass|
