# 测试用例归档

## 用例表

| 测试功能                        | 预置条件     | 输入 | 预期输出                 | 是否自动 | 测试结果 |
|-----------------------------| ------------ | ---- |----------------------| -------- | -------- |
| 不转码获取中文域名IP地址               | 设备正常运行 |      | 获取IP地址失败             | 是       | pass     |
| 转码获取中文域名IP地址                | 设备正常运行 |      | 获取到域名对应的IP地址         | 是       | pass     |

| 使用当前NetHandle对应的网络解析主机名和指定的IP类型以获取指定的IP地址 | 设备正常运行 |      | 以Promise形式返回指定的IP地址  | 是       | pass     |
