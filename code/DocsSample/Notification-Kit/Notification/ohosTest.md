# 事件通知测试用例

## 用例表

| 测试功能          | 预置条件              | 输入              | 预期输出                                        | 是否自动 | 测试结果 |
|---------------|-------------------|-----------------|---------------------------------------------|------|------|
| 弹窗请求通知授权      | 应用正常打开            | 点击请求通知授权        | 成功弹出授权弹窗                                    | Yes  | Pass |
| 再次申请通知授权      | 不涉及               | 点击再次申请授权        | 权限未打开时，弹出通知权限设置的半模态弹框；权限开启时，日志输出权限状态为true   | Yes  | Pass |
| 增加角标个数        | 通知权限开启            | 点击增加角标个数        | 主界面应用图标出现角标                                 | Yes  | Pass |
| 减少角标个数        | 通知权限开启            | 点击减少角标个数        | 主界面应用图标角标个数减少                               | Yes  | Pass |
| 设置角标个数为9      | 通知权限开启            | 点击设置角标个数为9      | 主界面应用图标角标个数显示为9                             | Yes  | Pass |
| 设置角标个数为8      | 通知权限开启            | 点击设置角标个数为8      | 主界面应用图标角标个数显示为8                             | Yes  | Pass |
| 角标设置个数反例      | 通知权限开启            | 点击角标设置个数反例      | 角标个数设置成功，但由于异步执行顺序原因，不符合预期（预期：11，实际：10或者11） | Yes  | Pass |
| 角标设置个数正例      | 通知权限开启            | 点击角标设置个数正例      | 角标个数设置成功，为11                                | Yes  | Pass |
| 清除通知          | 通知权限开启            | 点击清除通知          | 通知全部清除，角标不再显示                               | Yes  | Pass |
| 创建社交通信类型的通知渠道 | 通知权限开启            | 点击创建社交通信类型的通知渠道 | 日志打印创建社交通信类型通知渠道成功                          | Yes  | Pass |
| 查询社交通信类型的通知渠道 | 通知权限开启            | 点击查询社交通信类型的通知渠道 | 日志打印出通知渠道的设置情况                              | Yes  | Pass |
| 删除社交通信类型的通知渠道 | 通知权限开启            | 点击删除社交通信类型的通知渠道 | 日志打印删除社交通信类型通知渠道成功                          | Yes  | Pass |
| 发布普通文本类型通知    | 通知权限开启            | 点击发布普通文本类型通知    | 下拉框中弹出单行文本类型通知                              | Yes  | Pass |
| 发布多行文本类型通知    | 通知权限开启            | 点击发布多行文本类型通知    | 下拉框中弹出多行文本类型通知                              | Yes  | Pass |
| 查询系统是否支持进度条模板 | 通知权限开启            | 点击查询系统是否支持进度条模板 | 日志打印出是否支持，支持为true，不支持为false                 | Yes  | Pass |
| 发布进度条类型通知     | 通知权限开启，且系统支持进度条模板 | 点击发布进度条类型通知     | 下拉框中弹出进度条类型通知                               | Yes  | Pass |
| 发布不携带按钮的通知    | 通知权限开启            | 点击发布不携带按钮的通知    | 信息成功发布，点击该通知能够跳转到应用内                        | Yes  | Pass |
| 发布携带一个按钮的通知   | 通知权限开启            | 点击发布携带一个按钮的通知   | 信息成功发布，点击该通知能够显示按钮，点击按钮能够跳转到应用内             | Yes  | Pass |
| 发布携带两个按钮的通知   | 通知权限开启            | 点击发布携带两个按钮的通知   | 信息成功发布，点击该通知能够显示按钮，点击open_the_app按钮能够跳转到应用内 | Yes  | Pass |
| 更新通知          | 通知权限开启            | 点击更新通知          | 更新进度条类型通知成功，进度条由50%变为100%                   | Yes  | Pass |
| 取消通知          | 通知权限开启            | 点击取消通知          | 单行文本类型通知被取消，即下拉框中不显示单行文本类型                  | Yes  | Pass |
| 清除跨设备场景下的重复通知 | 通知权限开启，且多台设备同时在线  | 点击清除跨设备场景下的重复通知 | 会清除跨设备场景下的重复通知                              | Yes  | Pass |
