# 基础能力增强测试用例

## 用例表

| 测试功能                        | 预置条件     | 输入                        | 预期输出          | 是否自动 | 测试结果 |
|-----------------------------| ------------ |---------------------------|---------------|------| -------- |
| 点击订阅事件，点击两次发送事件             | 设备正常运行 |                           | 成功打印两次发送的事件信息 | Yes  | Pass     |
| 点击订阅事件，点击取消订阅，点击发送事件        | 设备正常运行 |                           | 不会再打印事件信息     | Yes  | Pass     |
| 点击订阅事件，点击发送事件，点击取消订阅，点击发送事件 | 设备正常运行 |                           | 收到一次事件信息      | Yes  | Pass     |
