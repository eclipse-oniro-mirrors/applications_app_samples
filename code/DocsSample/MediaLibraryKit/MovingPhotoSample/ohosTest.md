# Accessing and Managing Moving Photos测试用例归档

## 用例表

| 测试功能         | 预置条件  | 输入                 | 预期输出                     | 是否自动 | 测试结果 |
| ---------------- | -------- | -------------------- | ---------------------------- | -------- | -------- |
| 保存动态照片资源         | 进入"从媒体库获取(选择器)"界面       | 准备测试图像,然后点击"保存"按钮 | 动态照片成功保存到图库                 | 否       | Pass     |
| 获取动态照片对象         | 进入"从沙盒加载"界面       | 点击"示例"按钮 | 成功从媒体库获取动态照片对象                 | 否       | Pass     |
| 获取应用沙箱动态照片对象         | 进入"读取动态照片资产"界面       | 准备测试图像,然后点击"示例"按钮 | 成功从应用沙盒目录获取动态照片对象    | 否       | Pass     |
| 读取动态照片资源         | 进入"导出/读取动态照片内容"界面       | 准备测试图像,然后点击"示例"按钮 | 动态照片资产成功读取                 | 否       | Pass     |
