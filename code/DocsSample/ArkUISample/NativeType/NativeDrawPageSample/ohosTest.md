# Native Draw Page Sample 测试说明

## 测试环境

- 系统版本：OpenHarmony 6.0.2.56
- API 版本：API 22
- 设备：RK3568

## 测试用例

### 用例1：启动应用

**步骤：**
1. 安装应用
2. 点击应用图标启动

**预期结果：**
- 应用正常启动
- 显示主页面，包含"展示自定义绘制页面"按钮

### 用例2：打开自定义绘制页面

**步骤：**
1. 在主页面点击"展示自定义绘制页面"按钮

**预期结果：**
- 正常跳转到绘制页面
- 显示黄色背景的自定义绘制组件
- 组件中显示红色线条绘制效果

### 用例3：页面返回

**步骤：**
1. 在绘制页面点击返回按钮

**预期结果：**
- 正常返回到主页面
- 无崩溃或异常
