# AdjustLayout示例测试用例归档
## 用例表
| 测试功能       | 预置条件  | 输入                   | 预期输出         | 是否自动 | 测试结果 |
|------------| --- |----------------------|------------| --- |------|
| 拉起应用       |设备正常运行|                      | 成功拉起应用      | 是 | Pass |
| 设置主窗比例限制   |设备正常运行| 点击"开启主窗比例限制"按钮  | 主窗口保持宽高比1.5 | 是 | Pass |
| 显示/隐藏子窗   |设备正常运行| 点击"显示子窗/隐藏子窗"按钮 | 子窗口显示或隐藏    | 是 | Pass |
| 开启/关闭子窗拖拽大小 |设备正常运行| 点击拖拽大小控制按钮      | 子窗口可/不可拖拽调整大小 | 是 | Pass |
| 开启/关闭子窗拖拽位置 |设备正常运行| 点击拖拽位置控制按钮      | 子窗口可/不可拖拽移动位置 | 是 | Pass |
| 设置子窗windowLimits |设备正常运行| 输入尺寸限制参数并点击设置按钮 | 子窗口尺寸限制生效   | 是 | Pass |
