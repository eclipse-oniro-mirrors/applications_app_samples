# CustomCamera-master测试用例归档

## 用例表

| 测试功能     | 预置条件     | 输入                  | 预期输出     | 是否自动 | 测试结果 |
|----------|----------|---------------------|----------| -------- | -------- |
| 拉起应用          | 设备正常运行   |                     | 成功拉起应用   | 是       | Pass     |
| 申请相机权限       | 进入主界面    | 点击允许按钮              | 成功获取相机权限 | 是 | Pass |
| 模式切换动效   | 进入拍照模式页面      | 切换至录像模式           | 有模糊动效 | 是       | Pass     |
| 拍照模式动效   | 进入拍照模式页面      | 点击拍照                | 有闪黑动效 | 是       | Pass     |
| 拍照模式预览输出   | 进入拍照模式页面 | 查看相机预览是否正常 | 相机预览输出正常 | 是       | Pass     |
| 拍照功能          | 进入拍照模式页面    | 1. 点击拍照<br/>2. 点击拍照按钮           | 拍照图片保存至图库 |  是   | Pass |
| 录像模式预览输出   | 进入录像模式页面 | 点击录像模式页面，查看相机预览是否正常 | 相机预览输出正常 | 是       | Pass     |
| 录制功能          | 进入录像模式页面    | 1. 点击录像<br/>2. 点击录像开始按钮<br/>3. 一段时间后点击录像停止按钮     | 录像视频保存至图库 |  是   | Pass |
