# 音频会话控制测试用例归档

## 用例表

| 测试功能 | 预置条件 | 输入 | 预期输出 | 是否自动 |
|----------|----------|------|----------|----------|
| 拉起应用 | 设备正常运行 | - | 成功拉起应用，进入首页 | 是 |
| 激活音频会话 | 位于首页 | 点击 "设置场景+启用静音建议+激活焦点+注册监听" 按钮 | 音频会话激活成功，显示成功信息 | 是 |
| 设置会话级录音流静音提示 | 位于首页，麦克风权限已授权 | 点击 "设置会话级录音流静音提示" 按钮 | 成功启动录音流并调用`OH_AudioSessionManager_SetCaptureMuteHint`上报和解除静音提示，接口不实际触发静音，显示返回值 | 否 |
| 停用音频会话 | 音频会话已激活 | 点击 "注销焦点+注销监听" 按钮 | 音频会话停用成功，显示成功信息 | 是 |
