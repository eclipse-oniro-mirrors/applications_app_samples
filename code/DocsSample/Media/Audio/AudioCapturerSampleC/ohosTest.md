# 音频录制测试用例归档

## 用例表

| 测试功能 | 预置条件 | 输入 | 预期输出 | 是否自动 |
|----------|----------|------|----------|----------|
| 拉起应用 | 设备正常运行 | - | 成功拉起应用，进入 AudioCaptureDemo 首页 | 是 |
| 创建录制音频流（新版 API） | AudioCaptureDemo 页面 | 点击 "创建录制音频流" 按钮 | 成功创建音频录制流，使用新版 API 回调方式，显示创建成功信息（采样率 48000 Hz，声道数 2，回调方式：新版 API） | 是 |
| 创建录制流（使用空指针初始化） | AudioCaptureDemo 页面 | 点击 "创建录制流（使用空指针初始化）" 按钮 | 成功创建音频录制流，使用旧版回调方式 1（空指针初始化未使用的回调），显示创建成功信息 | 是 |
| 创建录制流（使用清零结构体初始化） | AudioCaptureDemo 页面 | 点击 "创建录制流（使用清零结构体初始化）" 按钮 | 成功创建音频录制流，使用旧版回调方式 2（清零结构体），显示创建成功信息 | 是 |
| 创建耳返录制流 | AudioCaptureDemo 页面 | 点击 "创建耳返录制流" 按钮 | 成功创建耳返录制流，时延模式为 FAST，用于耳返功能，显示创建成功信息 | 是 |
| 创建耳返播放流 | AudioCaptureDemo 页面 | 点击 "创建耳返播放流" 按钮 | 成功创建耳返播放流，时延模式为 FAST，用于耳返功能，显示创建成功信息 | 是 |
