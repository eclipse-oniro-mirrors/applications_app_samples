# 音频编创测试用例归档

## 用例表

| 测试功能 | 预置条件 | 输入 | 预期输出 | 是否自动 |
|----------|----------|------|----------|----------|
| 拉起应用 | 设备正常运行 | - | 成功拉起应用，进入首页 | 是 |
| 播放音频 | 位于首页 | 点击 "播放音频" 按钮 | 播放原始音频成功，显示成功信息 | 是 |
| 停止播放 | 音频正在播放 | 点击 "停止播放" 按钮 | 停止播放成功，显示成功信息 | 是 |
| 添加均衡器效果 | 位于首页 | 点击 "添加均衡器效果" 按钮 | 添加均衡器效果成功，显示成功信息 | 是 |
| 播放加均衡器效果后音频 | 已添加均衡器效果 | 点击 "播放加均衡器效果后音频" 按钮 | 播放加均衡器效果后音频成功，显示成功信息 | 是 |
| 音源分离 | 位于首页 | 点击 "音源分离" 按钮 | 音源分离成功，显示成功信息 | 是 |
| 播放伴奏 | 已完成音源分离 | 点击 "播放伴奏" 按钮 | 播放伴奏成功，显示成功信息 | 是 |
| 播放人声 | 已完成音源分离 | 点击 "播放人声" 按钮 | 播放人声成功，显示成功信息 | 是 |
| 混音与级联 | 已完成音源分离 | 点击 "混音与级联" 按钮 | 混音与级联成功，显示成功信息 | 是 |
| 播放混音 | 已完成混音与级联 | 点击 "播放混音" 按钮 | 播放混音成功，显示成功信息 | 是 |
| 播放实时预览 | 位于首页 | 点击 "播放实时预览" 按钮 | 播放实时预览成功，显示成功信息 | 是 |
| 播放空间渲染 | 已完成音源分离 | 点击 "播放空间渲染" 按钮 | 播放空间渲染成功，显示成功信息 | 是 |
| 音频格式转换 | 位于首页 | 点击 "格式转换" 按钮 | 格式转换成功，显示成功信息 | 是 |
| 调试 | 正在播放实时预览 | 点击 "调试" 按钮 | PrintInfo 测试成功，输出状态信息到文件 | 是 |
