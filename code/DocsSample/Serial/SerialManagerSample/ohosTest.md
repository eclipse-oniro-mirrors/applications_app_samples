# SerialManagerSample 测试用例归档

## 用例表

| 测试功能          | 预置条件         | 输入               | 预期输出                          | 是否自动 | 测试结果 |
|---------------|------------|------------------|-------------------------------|------|------|
| 拉起应用          | 	设备正常运行    | 		               | 成功拉起应用                        | 是    | Pass |
| 主页展示          | 设备正常运行      | 		               | 展示串口功能按钮                      | 是    | Pass |
| 主页按钮点击        | 位于主页，连接串口外设 | 	点击"getSerialPortList"      | 	日志显示获取的串口设备列表信息                 | 是    | Pass |
| 主页按钮点击        | 位于主页，已获取设备列表 | 点击"portInfo"       | 日志显示串口端口信息（portName、vendorId等）                   | 是    | Pass |
| 主页按钮点击        | 位于主页，已获取设备列表 | 点击"open"       | 日志显示打开串口成功                     | 是    | Pass |
| 主页按钮点击        | 位于主页，已打开串口设备 | 点击"write"         | 日志显示写入数据成功及写入长度                    | 是    | Pass |
| 主页按钮点击        | 位于主页，已打开串口设备 | 点击"onDataRead"         | 日志显示数据读取回调注册成功                    | 是    | Pass |
| 主页按钮点击        | 位于主页，已注册数据读取回调 | 点击"offDataRead"         | 日志显示取消数据读取回调成功                    | 是    | Pass |
| 主页按钮点击        | 位于主页，已打开串口设备 | 点击"flush"         | 日志显示刷新缓冲区成功                   | 是    | Pass |
| 主页按钮点击        | 位于主页，已打开串口设备 | 点击"drain"         | 日志显示等待输出数据传输完成成功                   | 是    | Pass |
| 主页按钮点击        | 位于主页，已打开串口设备 | 点击"setRts"         | 日志显示设置RTS信号成功                   | 是    | Pass |
| 主页按钮点击        | 位于主页，已打开串口设备 | 点击"getCts"         | 日志显示获取CTS信号状态成功                   | 是    | Pass |
| 主页按钮点击        | 位于主页，已打开串口设备 | 点击"sendBrk"         | 日志显示发送中断信号成功                   | 是    | Pass |
| 主页按钮点击        | 位于主页，已打开串口设备 | 点击"close"         | 日志显示关闭串口设备成功                   | 是    | Pass |
