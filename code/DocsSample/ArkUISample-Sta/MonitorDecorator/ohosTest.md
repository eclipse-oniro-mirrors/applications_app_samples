| @Monitor监听深层属性变化 | 设备正常运行 | 点击按钮 | 深层属性变化触发回调 | 是 | Pass |
| @Monitor在继承类场景监听属性变化 | 设备正常运行 | 点击按钮 | 基类和继承类监听回调先后触发 | 是 | Pass |
| @Monitor监听数组变化 | 设备正常运行 | 点击按钮 | 数组元素变化、整体赋值、长度变化触发回调 | 是 | Pass |
| @Monitor监听对象整体改变 | 设备正常运行 | 点击按钮 | 对象整体赋值时,变化的属性触发对应回调 | 是 | Pass |

| @Monitor通配符监听对象属性变化 | 设备正常运行 | 点击按钮 | propA、propB变化及对象整体赋值触发回调 | 是 | Pass |
| @Monitor通配符监听嵌套对象 | 设备正常运行 | 点击按钮 | 嵌套对象整体赋值、深层属性变化触发回调 | 是 | Pass |
| @Monitor通配符监听数组对象 | 设备正常运行 | 点击按钮 | 数组整体赋值、元素变化、push、shift等操作触发回调 | 是 | Pass |
| @Monitor通配符监听Date对象 | 设备正常运行 | 点击按钮 | Date API调用、整体赋值触发回调 | 是 | Pass |
| @Monitor通配符监听Map对象 | 设备正常运行 | 点击按钮 | Map API调用(set、delete、clear)触发回调 | 是 | Pass |
| @Monitor通配符监听Set对象 | 设备正常运行 | 点击按钮 | Set API调用(add、delete、clear)触发回调 | 是 | Pass |
| @Monitor监听深层属性变化-UIStyle示例 | 设备正常运行 | 点击按钮 | 监听info.value变化,根据变化百分比改变字体颜色和大小 | 是 | Pass |
