# @State装饰器测试用例归档
    	 
## 用例表

| 测试功能 | 预置条件 | 输入 | 预期输出 | 是否自动 | 测试结果 |
|---------|---------|------|---------|:-------:|:-------:|
| @State装饰@Observed类 | 设备正常运行 | 点击按钮修改属性 | UI刷新显示新值 | 是 | Pass |
| @State装饰简单类型 | 设备正常运行 | 点击按钮修改值 | UI刷新显示新值 | 是 | Pass |
| @State装饰Map类型 | 设备正常运行 | 点击按钮修改Map | UI刷新显示新Map内容 | 是 | Pass |
| @State装饰Set类型 | 设备正常运行 | 点击按钮修改Set | UI刷新显示新Set内容 | 是 | Pass |
| @State装饰Date类型 | 设备正常运行 | 点击按钮修改Date | UI刷新显示新Date内容 | 是 | Pass |
| @State装饰interface | 设备正常运行 | 点击按钮修改属性 | UI刷新显示新值 | 是 | Pass |
| @State支持联合类型 | 设备正常运行 | 点击按钮改变类型 | UI刷新显示新值和类型 | 是 | Pass |
