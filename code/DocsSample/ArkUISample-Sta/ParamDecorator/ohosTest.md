# @Param装饰器测试用例归档
  	 
## 用例表

根据提供的测试用例，以下是整理出的测试用例表格：

| 测试功能 | 预置条件 | 输入 | 预期输出 | 是否自动 | 测试结果 |
|---------|---------|------|---------|:-------:|:-------:|
| @Param装饰基本类型 | 设备正常运行 | 点击按钮 | count、message、flag值变化同步给子组件 | 是 | Pass |
| @Param装饰类对象 | 设备正常运行 | 点击按钮 | 对象整体赋值可观察，成员属性赋值依赖@Trace | 是 | Pass |
| @Param装饰数组 | 设备正常运行 | 点击按钮 | 数组整体和数组项变化可观察 | 是 | Pass |
| @Param装饰interface字面量 | 设备正常运行 | 点击按钮 | 字面量整体变化可观察，属性变化不可观察 | 是 | Pass |
| 从父组件到子组件变量传递与同步 | 设备正常运行 | 点击按钮 | 数据源修改同步给子组件@Param | 是 | Pass |
| 装饰Array类型变量 | 设备正常运行 | 点击按钮 | 数组整体和元素变化可观察，API操作可观察 | 是 | Pass |
| 装饰Date类型变量 | 设备正常运行 | 点击按钮 | Date整体和API操作变化可观察 | 是 | Pass |
| 装饰Map类型变量 | 设备正常运行 | 点击按钮 | Map整体和API操作变化可观察 | 是 | Pass |
| 装饰Set类型变量 | 设备正常运行 | 点击按钮 | Set整体和API操作变化可观察 | 是 | Pass |
| 联合类型 | 设备正常运行 | 点击按钮 | number和null切换可观察 | 是 | Pass |
