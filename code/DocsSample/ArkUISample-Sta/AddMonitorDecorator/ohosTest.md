| 重复监听示例 | 设备正常运行 | 点击按钮 | 重复监听可独立清除，不影响其他监听 | 是 | Pass |
| 同步监听模式 | 设备正常运行 | 点击按钮 | 同步模式回调立即触发，异步模式回调推迟 | 是 | Pass |
| 组件作为owner | 设备正常运行 | 点击按钮 | 组件作为owner传递，监听正常触发 | 是 | Pass |
| 配置path参数 | 设备正常运行 | 点击按钮 | path参数正确配置，监听回调路径正确 | 是 | Pass |
| 组件冻结功能 | 设备正常运行 | 点击按钮 | inactive页面回调推迟，active页面回调正常 | 是 | Pass |

| 监听数组元素和长度 | 设备正常运行 | 点击按钮 | 数组元素和长度变化可观察 | 是 | Pass |

| 动态取消监听 | 设备正常运行 | 点击按钮 | 监听可动态添加和清除 | 是 | Pass |
