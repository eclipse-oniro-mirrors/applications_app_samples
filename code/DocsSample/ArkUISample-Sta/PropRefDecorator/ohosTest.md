# @PropRef装饰器测试用例归档
    	 
## 用例表

| 测试功能 | 鐐置条件 | 输入 | 鐐期输出 | 是否自动 | 测试结果 |
|---------|---------|------|---------|:-------:|:-------:|
| @PropRef简单类型变化观察 | 设备正常运行 | 点击按钮修改父组件值 | 子组件同步更新 | 是 | Pass |
| @PropRef父子组件单向同步 | 设备正常运行 | 点击按钮修改值 | 父组件值不变，子组件值变化 | 是 | Pass |
| @PropRef获得父组件数据源引用 | 设备正常运行 | 点击按钮修改属性 | 父组件属性也会修改 | 是 | Pass |
| @PropRef装饰Array类型 | 设备正常运行 | 点击按钮修改数组 | UI刷新显示新数组内容 | 是 | Pass |
| @PropRef装饰Map类型 | 设备正常运行 | 点击按钮修改Map | UI刷新显示新Map内容 | 是 | Pass |
| @PropRef装饰Set类型 | 设备正常运行 | 点击按钮修改Set | UI刷新显示新Set内容 | 是 | Pass |
| @PropRef装饰Date类型 | 设备正常运行 | 点击按钮修改Date | UI刷新显示新Date内容 | 是 | Pass |
