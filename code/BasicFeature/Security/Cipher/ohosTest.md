# Cipher 测试用例归档

## 用例表

|测试功能|预置条件|输入|预期输出|测试结果|
|--------------------------------|--------------------------------|--------------------------------|--------------------------------|--------------------------------|
|拉起应用|设置正常运行|             |成功拉起应用|Pass|
|跳转加密页面|成功拉起应用|点击加密按钮|成功跳转至加密页面|Pass|
|加密算法选择|成功跳转至加密页面|点击下拉选择，选择AES加密算法|Encryption Algorithm变为AES|Pass|
|加密内容|选择了加密算法|不输入内容，点击加密按钮|弹出toast提示：This message is null|Pass|
|加密内容|选择了加密算法|输入要加密的内容，点击加密按钮|下方文本框显示加密内容|Pass|
|重置加密内容|文本框内有加密内容|点击重置按钮|文本框内加密内容被清空|Pass|
|返回首页|加密操作全部执行完毕|点击返回|成功返回首页|Pass|
|跳转解密页面|成功返回首页|点击解密按钮|成功跳转至解密页面|Pass|
|解密算法选择|成功跳转至解密页面|点击下拉选择，选择AES解密算法|Decrypt Algorithm变为AES|Pass|
|解密内容|选择了解密算法|不输入内容，点击解密按钮|弹出toast提示：This message is null|Pass|
|解密内容|选择了解密算法|输入要解密的内容，点击解密按钮|下方文本框显示解密内容|Pass|
|重置解密内容|文本框内有解密内容|点击重置按钮|文本框内解密内容被清空|Pass|
|返回首页|解密操作全部执行完毕|点击返回|成功返回首页|Pass|
