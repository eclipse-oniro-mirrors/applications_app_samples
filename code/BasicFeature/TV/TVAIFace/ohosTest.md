# AIFace 测试用例归档

## 用例表

| 测试功能                              | 预置条件                                | 输入                                      | 预期输出                               | 是否自动 | 测试结果 |
|-----------------------------------|-------------------------------------|-----------------------------------------|------------------------------------|------|------|
| 图片人脸检测 | 1. 需在真机测试 <br/> 2. 构建并安装测试hap <br/> 3. 设备中保存有人脸的照片 <br/> | 点击人脸图片识别，选择照片                           | 识别出人脸的位置 | 否    | Pass |
| 摄像头人脸检测 | 1. 需在真机测试 <br/> 2. 构建并安装测试hap <br/> 3. TV设备连接摄像头 | 点击人脸摄像头识别，显示当前选择的摄像头画面                  | 识别出人脸的位置 | 否    | Pass |
| 设备连接验证 |  1. 需在真机测试 <br/> 2. 构建并安装测试hap <br/> 3. 同一网络下存在相同设备 <br/> | 1、点击分布式验证 <br/> 2、搜索设备<br/> 3、连接设备<br/> | 能够看到连接设备的画面，识别出人脸位置 | 否    | Pass |
