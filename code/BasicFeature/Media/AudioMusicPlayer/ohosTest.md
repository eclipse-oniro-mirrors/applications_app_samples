# 基于OHAudio播放PCM音频测试用例

## 用例表

| 测试功能 | 预置条件 | 输入 | 预期输出 | 是否自动 | 测试结果 |
| --- | --- | --- | --- | --- | --- |
| 拉起应用 | 设备正常运行 | | 成功拉起应用，进入首页 | 是 | Pass |
| 主页展示 | 设备正常运行 | | 展示音乐播放界面，包含播放按钮、进度条、歌曲信息 | 是 | Pass |
| 播放功能 | 位于主页 | 点击底部播放按钮 | 音乐开始播放，播放按钮变为暂停状态 | 是 | Pass |
| 暂停功能 | 音乐正在播放 | 点击暂停按钮 | 音乐暂停播放，暂停按钮变为播放状态 | 是 | Pass |
| 切歌功能-下一首 | 位于主页 | 点击下一首按钮 | 切换到下一首歌曲，歌曲信息更新 | 是 | Pass |
| 切歌功能-上一首 | 位于主页 | 点击上一首按钮 | 切换到上一首歌曲，歌曲信息更新 | 是 | Pass |
| 进度跳转功能 | 位于主页 | 拖动进度条滑块 | 播放进度跳转到指定位置 | 是 | Pass |
| 循环模式切换 | 位于主页 | 点击循环模式图标 | 依次切换"顺序播放"、"单曲循环"、"随机播放"模式 | 是 | Pass |
| 歌单列表展示 | 位于主页 | 点击歌单列表图标 | 弹出歌单列表，显示所有歌曲 | 是 | Pass |
| 歌单列表切歌 | 歌单列表已打开 | 点击歌曲名称 | 切换到指定歌曲播放 | 是 | Pass |
| 静音播放 | 位于主页 | 点击静音图标 | 开启静音模式，静音图标变为激活状态 | 是 | Pass |
| 取消静音 | 静音模式已开启 | 点击静音图标 | 关闭静音模式，静音图标恢复正常状态 | 是 | Pass |
| 倍数设置 | 位于主页 | 点击倍数设置图标 | 弹出倍数选择面板，显示0.5x-2.0x选项 | 是 | Pass |
| 倍数调整 | 倍数面板已打开 | 选择指定倍数 | 播放速度调整为指定倍数 | 是 | Pass |
| 音量设置 | 位于主页 | 点击音量设置图标 | 弹出音量调节滑块 | 是 | Pass |
| 音量调整 | 音量面板已打开 | 拖动音量滑块 | 音量大小调整为指定值 | 是 | Pass |
| 收藏功能 | 位于主页 | 点击"爱心"图标 | 歌曲变为已收藏状态，爱心图标变为红色 | 是 | Pass |
| 取消收藏 | 歌曲已收藏 | 点击"爱心"图标 | 取消收藏状态，爱心图标恢复正常 | 是 | Pass |
| 后台播放 | 音乐正在播放 | 切换应用到后台 | 音乐继续播放，通知栏显示播放控制 | 是 | Pass |
| 播控中心交互 | 后台播放中 | 点击通知栏控制按钮 | 应用响应控制命令，执行播放/暂停/切歌操作 | 是 | Pass |
