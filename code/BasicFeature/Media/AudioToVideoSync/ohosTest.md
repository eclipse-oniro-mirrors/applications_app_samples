# 实现音画同步播放效果测试用例

| 测试功能         | 预置条件                      | 输入            | 预期输出                                                | 是否自动 | 测试结果 |
|--------------|---------------------------|---------------|-----------------------------------------------------|------|------|
| 相册中的视频正常播放  | 	启动应用 | 	点击播放按钮，选择图库中的视频，开始播放	 | 视频正常播放 | 否    | Pass |
