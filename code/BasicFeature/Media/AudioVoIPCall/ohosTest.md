# AudioVoIPCallSample 测试用例归档

## 用例表

| 测试功能    | 预置条件         | 输入             | 预期输出                  | 是否自动 | 测试结果 |
|---------|--------------|----------------|-----------------------|------|------|
| 拉起应用    | 设备正常运行      | 		             | 成功拉起应用                | 是    | Pass |
| 主页展示    | 设备正常运行      | 		             | 展示模拟来电和模拟去电的入口按钮 | 是    | Pass |
| 允许权限弹窗| 位于主页 | 点击主叫/被叫按钮|弹出允许权限选择窗口 | 是    |Pass|
| 允许权限| 允许权限已弹窗 | 点击权限弹窗允许按钮 |授权后成功进入来电/去电页面 | 是    |Pass|
| 主页按钮点击  | 位于主页        | 点击"被叫" | 跳转"来电"页面      | 是    | Pass |
| 来电页面展示  | 位于来电页面    | 查看页面内容   | 展示来电标题、接听和拒绝按钮          | 是    | Pass |
| 来电铃声播放  | 位于来电页面    | 进入页面      | 自动播放来电铃声            | 是    | Pass |
| 来电接听    | 位于来电页面    | 点击"接听"按钮  | 跳转至通话中页面          | 是    | Pass |
| 返回主页    | 位于来电页面    | 点击"拒绝"按钮   | 回到主页，停止铃声播放                 | 是    | Pass |
| 主页按钮点击  | 位于主页        | 点击"主叫" | 跳转"去电"页面             | 是    | Pass |
| 去电页面展示  | 位于去电页面    | 查看页面内容   | 展示去电标题、对方接听和对方拒绝按钮          | 是    | Pass |
| 去电接听    | 位于去电页面    | 点击"对方接听"按钮  | 跳转至通话中页面          | 是    | Pass |
| 返回主页    | 位于去电页面    | 点击"对方拒绝"按钮   | 回到主页                 | 是    | Pass |
| 通话中页面展示  | 位于通话中页面    | 查看页面内容   | 展示通话中标题、AVCastPicker设备选择图标、麦克风图标、扬声器图标、挂断按钮          | 是    | Pass |
| AVCastPicker设备切换  | 位于通话中页面    | 点击AVCastPicker图标  | 弹出音频输出设备选择列表（听筒、扬声器、蓝牙）          | 是    | Pass |
| 切换至扬声器  | 位于通话中页面    | 在设备列表中选择扬声器  | 当前设备显示为扬声器，音频输出切换至扬声器          | 是    | Pass |
| 切换至听筒  | 位于通话中页面    | 在设备列表中选择听筒  | 当前设备显示为听筒，音频输出切换至听筒          | 是    | Pass |
| 麦克风静音  | 位于通话中页面    | 点击麦克风图标  | 麦克风图标切换为静音状态，录音静音          | 是    | Pass |
| 麦克风取消静音  | 位于通话中页面    | 点击静音状态的麦克风图标  | 麦克风图标切换为正常状态，录音恢复          | 是    | Pass |
| 扬声器静音  | 位于通话中页面    | 点击扬声器图标  | 扬声器图标切换为静音状态，播放静音          | 是    | Pass |
| 扬声器取消静音  | 位于通话中页面    | 点击静音状态的扬声器图标  | 扬声器图标切换为正常状态，播放恢复          | 是    | Pass |
| 挂断通话    | 位于通话中页面    | 点击挂断按钮  | 返回主页，释放音频资源          | 是    | Pass |
| 来电超时弹窗  | 位于来电页面    | 等待超时时间  | 自动停止铃声播放，弹出"响铃超时"提示框          | 是    | Pass |
| 确认超时弹窗  | 位于来电页面    | 点击超时弹窗的"确认"按钮  | 返回主页          | 是    | Pass |
| 提前停止铃声  | 位于来电页面    | 在超时前点击"拒绝"按钮  | 停止铃声播放，返回主页，不再弹出超时提示框          | 是    | Pass |
