# 语法能力增强测试用例

|----------------------|-------------------------------------|-------------------------|------------|------|------|
| 支持UI中使用Switch/Case语法 | 1. 需在真机测试 <br/> 2. 构建并安装测试hap <br/> 3.启动测试hap| 左右滑动界面，点击界面中数字或下一步      | 正常跳转步骤     | 否    | Pass |
