# OrangeShopping测试用例归档

|通过Tab切换不同页面|位于首页|1.点击新品Tab<br/>2.点击购物车Tab<br/>3.点击用户中心Tab<br/>4.点击首页Tab|1.切换到新品页面<br/>2.切换到购物车页面<br/>3.切换到用户中心页面<br/>4.切换到首页页面 |Pass|
|播放视频|位于首页| 1.点击商品<br/>2.点击播放键 |1.打开对应商品详情页<br/>2.视频正常播放|Pass|
|全屏播放|位于商品详情页，视频正在播放| 1.点击正在播放的视频组件<br/>2.点击全屏时的视频界面<br/>3.点击播放键 |1.切换到全屏视频页<br/>2.视频暂停播放并出现暂停按钮<br/>3.视频恢复播放|Pass|
|添加购物车|位于首页| 1.点击商品<br/>2.点击“添加购物车”按钮<br/>3.点击“购物车”图标 |1.进入到对应商品的详情页<br/>2.添加对应的商品到购物车中<br/>3.检查商品已经出现在购物车中|Pass|

|首页设计结构|位于首页|检查顶部扫码入口、搜索框、业务导航、推荐分类、商品卡片|首页核心信息架构和关键入口均存在|Pass|
|首页推荐分类切换|位于首页|依次点击精选、新品、实惠、全部推荐分类|推荐分类切换后商品列表和分类入口仍保持可用|Pass|
|新品页设计结构|位于首页|点击新品Tab，检查轮播、分类筛选、商品列表，并切换分类|新品页首屏、分类入口和商品卡片均正常显示|Pass|
|购物车空态/推荐态|位于购物车页|检查购物车标题、编辑入口、自营店铺区、猜你喜欢区、底部Tab|购物车基础结构和推荐区可见，底部导航可继续操作|Pass|
|个人中心设计结构|位于个人中心页|检查设置、扫码、用户信息、订单、权益、服务、直播区块|个人中心主要区块和服务入口完整显示|Pass|

|商品收藏交互|位于商品详情页|点击收藏入口，再次点击取消收藏|页面保持在商品详情上下文，收藏入口可重复操作|Pass|
|降价提醒交互|位于商品详情页|点击“降价提醒”|出现订阅反馈或保留原操作入口，商品详情上下文不丢失|Pass|

|详情页回首页|位于商品详情页|点击底部“首页”入口|返回首页后顶部搜索和商品列表可见|Pass|
|详情页直播入口|位于商品详情页|点击“直播中”组件并返回|直播页正常打开，返回后仍在商品详情页|Pass|
|详情页视频交互|位于商品详情页|点击播放、进入全屏、暂停、返回|预览图播放后隐藏，全屏页可暂停并返回详情页|Pass|
|购物闭环流程|位于首页|进入商品、加入购物车、进入购物车、返回首页|商品加入购物车成功，购物车和首页之间可完成闭环跳转|Pass|
|首页业务导航切换|位于首页|依次点击视频、运动、家电等业务导航|切换业务导航后搜索、分类和商品列表主结构保持可用|Pass|

|详情服务评价区域|位于商品详情页|检查服务承诺、评价标题、好评率和好评文案|详情页服务与评价信息完整显示|Pass|
|个人中心直播区|位于个人中心页|滚动到直播推荐区，检查标题、查看更多和直播标签|个人中心直播推荐区域可访问且内容完整|Pass|
