# International测试用例归档

### 用例表

| 测试功能   | 预置条件     | 输入      | 预期输出                 | 测试结果 |
|--------|----------|---------|----------------------|------|
| 拉起应用   | 设备正常运行   | 无       | 成功拉起应用               | Pass |
| 主页按钮点击 | 位于主页     | 点击日期和时间 | 跳转日期和时间页             | Pass |
| 日期和时间页 | 位于日期和时间页 | 点击时区    | 跳转时区页                | Pass |
| 设置时区   | 位于时区页    | 点击新西兰   | 回到日期和时间页，底部时区显示新西兰时间 | Pass |
