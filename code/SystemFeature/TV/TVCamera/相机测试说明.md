# Camera测试说明

## 应用概述

本应用是OpenHarmony camera相机应用，包含完整的拍照、录像、多相机切换等功能。

## 测试范围

### 功能测试

- 相机拍照功能测试
- 相机录像功能测试
- 多相机切换测试
- 相机设置功能测试
- 悬浮窗功能测试

### 测试环境

- OpenHarmony TV设备
- Camera硬件模块
- 触摸屏/遥控器交互

## 测试用例说明

### 单元测试

测试代码位于 `entry/src/ohosTest/ets/test/` 目录下，使用Hypium测试框架。

### 集成测试

集成测试覆盖主要业务流程，确保端到端功能正常。

## 测试报告

测试结果通过ohosTest模块输出，使用Hilog记录测试日志。
