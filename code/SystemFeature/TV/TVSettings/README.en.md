## Introduction
Settings is a preinstalled system application in OpenHarmony. It provides users with an interactive interface for configuring system properties such as system sound, screen brightness, wired networks, wireless networks, Bluetooth connections, device name, time, date, and other system settings.

## TV Features
OpenHarmony system applications can adapt their component form to different products and screen types. In the TV form factor, Settings provides the following features:
- TV-style UX
- Supports selecting the HDMI output mode
- Supports selecting the next-generation audio fallback output mode
- Supports selecting the SPDIF output mode
- Supports configuring wired networks
- Supports network management
- Supports adjusting hue, contrast, and saturation
- Supports selecting the HDR mode
- Supports configuring CEC and HDCP
- Supports setting the display area
- Supports setting the display standard
- Supports switching between landscape and portrait orientation

## Constraints
- Development environment
    - **IDE**: DevEco Studio (version 4.1.0.400)
    - **SDK**: Full SDK (version 3.2.10.10)

- Language version
    - [eTS](https://gitee.com/openharmony/docs/blob/master/zh-cn/application-dev/quick-start/start-with-ets.md)

- Recommendation
    - It is recommended to use the signing configuration provided in this project. The signing files are located in the `signature` directory.
