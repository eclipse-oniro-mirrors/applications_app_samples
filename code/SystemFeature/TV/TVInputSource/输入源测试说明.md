# 输入源测试说明

## 应用概述

本工程是 OpenHarmony TV 输入源示例应用（`TVInputSource`），覆盖 HDMI / AV / DTV 等输入源展示、切换、信号检测、页面联动与事件发布能力。

## 测试范围

### 功能测试

- 输入源列表展示与切换逻辑
- HDMI 页面信号状态处理与回退策略
- DTV 初始化、搜索与频道相关逻辑
- 首页焦点、路由与生命周期联动
- 播放控制、事件发布与状态恢复逻辑

### 测试环境

- OpenHarmony TV设备
- 输入源能力模块（HDMI/AV/DTV）
- 触摸屏/遥控器交互

## 测试用例说明

### 单元测试

测试代码位于 `entry/src/ohosTest/ets/test/` 目录，使用 Hypium 框架。
统一入口文件为 `entry/src/ohosTest/ets/test/List.test.ets`，由该文件聚合并执行所有启用的 `*.test.ets` 用例。

### 集成测试

集成测试覆盖输入源主流程（首页进入、输入源切换、信号处理、页面回退），用于验证关键链路端到端行为。

## 测试报告

测试结果通过 `ohosTest` 模块输出，日志通过 Hilog 记录。
建议在本地构建目录结合 `hvigor` / DevEco Studio 的测试输出查看失败用例与断言信息。
