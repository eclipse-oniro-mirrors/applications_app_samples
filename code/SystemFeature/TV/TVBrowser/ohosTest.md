# TVBrowser 测试用例归档

## 用例表

| 测试功能 | 预置条件 | 输入 | 预期输出 | 是否自动 | 测试结果 |
|---|---|---|---|---|---|
| 拉起应用 | 1. 连接 Wi-Fi/网络 <br/> 2. 构建并安装应用 | 通过桌面图标启动 | 成功拉起应用并进入首页 | 否 | Pass |
| 首页快捷入口跳转 | 位于首页 | 点击默认图标/快捷入口 | 跳转到对应网页 | 否 | Pass |
| 返回首页 | 已进入特定网页 | 点击返回/主页按钮 | 返回到首页 | 否 | Pass |
| 输入网址跳转 | 位于首页 | 文本框输入地址，点击跳转按钮 | 成功跳转到对应页面 | 是 | Pass |
| 前进/后退导航 | 已有浏览记录 | 点击前进/后退按钮 | 页面可正常前进/后退 | 是 | Pass |
| 新增页面 | 设备支持多页面（如平板形态） | 点击加号按钮 | 新建一个页面 | 是 | Pass |
