## 应用功能模块清单

| 模块名称 | 功能描述 | 设备支持 |
|---|---|---|
| 网页浏览 (WebView) | 基于ArkWeb的网页浏览功能 | rk3568、大屏 |
| 多标签页管理 (BrowserTabs) | 多网页标签页创建、切换与删除功能 | rk3568、大屏 |
| 导航栏控制 (TabletTitle) | 前进、后退、刷新、主页等导航功能 | rk3568、大屏 |
| 网络状态检测 (NetworkState) | WiFi连接状态检测与网络异常处理 | rk3568、大屏 |
| 设备适配 (DeviceAdaptation) | 不同设备类型（手机、平板）的界面适配 | rk3568、大屏 |
