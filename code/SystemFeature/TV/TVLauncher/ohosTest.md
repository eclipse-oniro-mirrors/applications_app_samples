# TVLauncher 测试用例归档

## 用例表

| 测试功能 | 预置条件 | 输入 | 预期输出 | 是否自动 | 测试结果 |
|---|---|---|---|---|---|
| 启动器主界面启动 | 1. 需在真机测试 <br/> 2. 构建并安装 launcher hap | 启动应用 | 成功进入 TVLauncher 主界面，无崩溃 | 否 | Pass |
| 焦点移动与遥控按键 | 1. 设备连接遥控器或输入设备 <br/> 2. 启动器已启动 | 按上下左右键移动焦点 | 焦点在卡片与导航区域间正确移动，样式高亮正确 | 否 | Pass |
| 应用打开与返回 | 1. 主界面存在可启动应用 | 选中应用并按确认键启动，再按返回键 | 应用可正常打开并返回 TVLauncher | 否 | Pass |
| 最近任务展示 | 1. 至少启动过一个应用 | 进入最近任务页 | 展示最近任务列表，切换与退出操作正常 | 否 | Pass |
| 设置项持久化 | 1. 完成一次设置项修改 | 重启设备或重启应用后查看设置 | 配置保持生效，数据持久化正确 | 否 | Pass |
