# TVDemandPlayer 测试用例归档

## 用例表

| 测试功能                              | 预置条件                                | 输入                                      | 预期输出                               | 是否自动 | 测试结果 |
|-----------------------------------|-------------------------------------|-----------------------------------------|------------------------------------|------|------|
| 本地视频播放 | 1. 需在真机测试 <br/> 2. 构建并安装测试hap <br/> 3. 设备中存在视频文件 <br/> | 点击本地视频，选择视频文件                           | 正常播放视频 | 否    | Pass |
| HTTP视频播放 | 1. 需在真机测试 <br/> 2. 构建并安装测试hap <br/> 3. 设备连接网络 | 点击网络视频，输入URL地址                  | 正常播放网络视频 | 否    | Pass |
| 视频播放控制 | 1. 需在真机测试 <br/> 2. 构建并安装测试hap <br/> 3. 正在播放视频 <br/> | 点击播放/暂停/快进/快退按钮 | 视频按指令进行播放控制 | 否    | Pass |
| 设备连接验证 |  1. 需在真机测试 <br/> 2. 构建并安装测试hap <br/> 3. 同一网络下存在相同设备 <br/> | 1、点击分布式验证 <br/> 2、搜索设备<br/> 3、连接设备<br/> | 能够进行视频流转播放 | 否    | Pass |
| DLNA投屏 | 1. 需在真机测试 <br/> 2. 构建并安装测试hap <br/> 3. 同一网络下存在DLNA设备 <br/> | 点击DLNA投屏，选择设备 | 视频投屏到DLNA设备播放 | 否    | Pass |
