# 页面加载效果实现案例测试用例

## 用例表

| 测试功能         | 预置条件                      | 输入            | 预期输出                                                | 是否自动 | 测试结果 |
|--------------|---------------------------|---------------|-----------------------------------------------------|------|------|
| 输入密码数量不足  | 	启动应用 | 手指滑动连接圆点0，1，2 | 提示密码输入数量不足，选中圆点显示错误态效果 | 否 | Pass |
| 输入错误密码 | 启动应用 | 手指滑动连接圆点0，1，2，3 | 提示密码输入错误，选中圆点显示错误态效果 | 否 | Pass |
| 输入正确密码 | 启动应用 | 手指滑动连接圆点0，1，2，5，8 | 提示密码输入正确，选中圆点显示正确态效果 | 否    | Pass |
